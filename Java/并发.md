# 1. 什么是进程和线程？
  
* **进程**
    进程是系统运行程序的最小单位，系统运行一个程序就相当于创建一个进程。
* **线程**
  线程与进程相似，但是线程是比进程更小的执行单位。一个进程可以产生多个线程。线程可以共享进程的堆和方法区资源，每个线程都拥有自己的**程序计数器、虚拟机栈和本地方法栈**，所以系统产生一个线程或者线程之间切换时，负担比进程要小。
